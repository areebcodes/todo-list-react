# To-Do List App

A simple To-Do List application built with React and Vite.

## Features

- Add new tasks
- Delete tasks
- Reorder tasks up and down

## Tech Stack

- React
- JavaScript
- CSS
- Vite

## Getting Started

1. Clone the repository
```
git clone https://github.com/areebcodes/todo-list-react.git
```

2. Install dependencies
```
npm install
```

3. Run the app
```
npm run dev
```

4. Open your browser and go to `http://localhost:5173`
